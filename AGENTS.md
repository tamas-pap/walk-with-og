# AGENTS.md

## Cursor Cloud specific instructions

This repository is **specification-only**. It contains a product design spec for an
audio-first walking-companion app ("Walk with Mungo" / "Og") as Markdown documents
under `spec/`. There is currently:

- No application code, package manifests, or lockfiles
- No build system, tests, linters, or CI configured
- No services to run and no dependencies to install

There is therefore nothing to build, run, or test as an application. Work in this repo
is editing/authoring the Markdown specs in `spec/`. No environment setup is required;
the startup update script is intentionally a no-op until runnable code is added.

If/when application code is introduced, update this section with the real
install/build/test/run commands and remove this no-op note.
